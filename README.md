# CareerNest-Job-Portal
CareerNest is a full-stack Job Portal built with Flask, SQLite, HTML, CSS, and JavaScript. Candidates can browse and apply for jobs, recruiters can post jobs and view applications, and admins can monitor everything with secure role-based authentication.
